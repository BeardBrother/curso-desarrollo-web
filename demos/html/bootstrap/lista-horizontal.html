Listas de definición horizontal

Agente de usuario
EN HTML el agente de usuario es el dispositivo que interpreta el documento HTML.
Scripting del lado del cliente
Cuando hablamos de scripting del lado del cliente nos referimos a una categoria de programas de ordenador en la web que son ejecutados en el lado del cliente, por ejemplo, el navegador web.
Árbol del Documento
Es el árbol de elementos que se extrae del documento fuente.
